Class: Combo

Combo

Combo represents the physical combination of a single key and any meta keys that ultimately are used to trigger a keybinding. It is at the Combo level that we match a configured keybinding with the current set of pressed keys.

new Combo(key, meta)

Creates a new Combo from a key code and array of meta key codes

Parameters:
Name Type Description
key Key

The primary Key for this Combo.
meta array

An array of meta Keys required for this Combo to be activated.
Source:
  • keys.js, line 385
Example
 var combo = new Combo(Key.A, [ Key.CTRL, Key.SHIFT ]);
